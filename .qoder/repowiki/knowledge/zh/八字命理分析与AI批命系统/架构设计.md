- 采用 React + Vite + TypeScript 技术栈，通过 `lunar-typescript` 库处理农历与干支转换。
- 核心逻辑分层：`types/` 定义领域模型（如 `BaziResult`），`utils/` 封装纯函数计算逻辑（五行、十神、大运），`hooks/` 管理状态与副作用（`useBazi` 协调计算，`useAI` 处理流式请求）。
- 组件化展示：`App.tsx` 作为布局容器，通过 Tab 切换展示 `BaziChart`（命盘）、`WuxingChart`（五行）、`DayunList`（大运）及 `Analysis`（AI 批命）。
- AI 集成：`Analysis` 组件结合 `marked` 与 `DOMPurify` 渲染 Markdown 格式的流式响应，配置信息持久化于 `localStorage`。
